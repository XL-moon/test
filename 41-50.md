## 31-40题
### 三十一、连续子数组的最大和
#### 题目及思路
```java<br>
题目：
求连续子数组（包含负数）的最大和
思路：
若和小于0，则将最大和置为当前值，否则计算最大和。
```
#### 代码
```java<br>
代码实现：

    public int FindGreatestSumOfSubArray(int[] array) {
        if (array == null || array.length == 0)
            return 0;
        int cur = array[0];
        int greast = array[0];

        for (int i = 1; i < array.length; i++) {
            if (cur < 0) {
                cur = array[i];
            }else {
                cur += array[i];
            }

            if (cur > greast) {
                greast = cur;
            }
        }
        return greast;
    }
```
