### 四十一、和为s的连续整数序列
#### 题目1
```java<br>
题目：
输出所有和为S的连续正数序列。序列内按照从小至大的顺序，序列间按照开始数字从小到大的顺序
思路：
定义两个指针，分别递增，寻找和为s的序列。
```
#### 代码
```java<br>
代码实现：

   public ArrayList<ArrayList<Integer>> FindContinuousSequence(int sum) {
        ArrayList<ArrayList<Integer>> arrayList = new ArrayList<>();
        ArrayList<Integer> list = new ArrayList<>();
        if (sum < 3)
            return arrayList;
        int small = 1;
        int big = 2;
        while (small < (sum + 1) / 2) {
            int s = 0;
            for (int i = small; i <= big; i++) {
                s += i;
            }
            if (s == sum) {
                for (int i = small; i <= big; i++) {
                    list.add(i);
                }
                arrayList.add(new ArrayList<>(list));
                list.clear();
                small++;
            } else {
                if (s > sum) {
                    small++;
                } else {
                    big++;
                }
            }
        }
        return arrayList;
    }
```
#### 题目2
```java<br>
题目：
输入一个递增排序的数组和一个数字S，在数组中查找两个数，是的他们的和正好是S，如果有多对数字的和等于S，输出两个数的乘积最小的。
思路：
定义两个指针，分别从前面和后面进行遍历。间隔越远乘积越小，所以是最先出现的两个数乘积最小
```
#### 代码
```java<br>
代码实现：

public ArrayList<Integer> FindNumbersWithSum(int [] array,int sum) {
    ArrayList<Integer> list = new ArrayList<>();
    if (array == null )
        return list;
    int left = 0;
    int right = array.length - 1;

    while (left < right) {
        int s = array[left] + array[right];
        if (s == sum) {
            list.add(array[left]);
            list.add(array[right]);
            return list;
        }else {
            if (s > sum) {
                right--;
            }else {
                left++;
            }
        }
    }
    return list;
}
```

### 四十二、翻转字符串
#### 题目1
```java<br>
题目：
翻转字符串
思路：
先将整个字符串翻转，然后将每个单词翻转。
```
#### 代码
```java<br>
代码实现：

public String ReverseSentence(String str) {
    if (str == null || str.length() == 0)
        return str;
    if (str.trim().length() == 0)
        return str;
    StringBuilder sb = new StringBuilder();
    String re = reverse(str);
    String[] s = re.split(" ");
    for (int i = 0; i < s.length - 1; i++) {
        sb.append(reverse(s[i]) + " ");
    }
    sb.append(reverse(s[s.length-1]));
    return String.valueOf(sb);
}

public String reverse(String str) {
    StringBuilder sb = new StringBuilder();
    for (int i = str.length() - 1; i >= 0 ; i--) {
        sb.append(str.charAt(i));
    }
    return String.valueOf(sb);
}
```
#### 题目2
```java<br>
题目：
对于一个给定的字符序列S，请你把其循环左移K位后的序列输出
思路：
拼接或反转三次字符串
```
#### 代码
```java<br>
代码实现：

public String LeftRotateString(String str,int n) {
    if (str == null || str.length() == 0)
        return str;
    String s1 = reverse(str.substring(0,n));
    String s2 = reverse(str.substring(n,str.length()));
    return reverse(s2)+reverse(s1);
}
```
